# Mini-Projet NLP

Projet de mini-projet NLP pour ENSA Al Hoceima.

## Structure du projet

- `data/` : corpus et données de base
- `src/` : code source Python
  - `src/exploration/` : exploration des Transformers
  - `src/rag/` : système RAG
- `notebooks/` : notebooks Jupyter
- `models/` : modèles locaux et index enregistrés
- `reports/` : rapport final et documents
- `tests/` : scripts de test
- `plan/` : plan de projet et jalons
- `req/requirements.txt` : dépendances Python

## Installation

```powershell
.\env\Scripts\Activate.ps1
python -m pip install -r req/requirements.txt
```
